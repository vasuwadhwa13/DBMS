SQL Query Example

Description

This query stores the marks of students and filters the top 3 students based on performance.

Criteria: A student must have:

  • Aggregate more than 60%
  • No failed subjects (marks < 30)
  • No subject marked as 'absent'

Example Scenario

  • 4 students are failing overall
  • 2 students marked absent in one subject
  • One absentee has the highest aggregate
  • Includes aggregate and percentage columns
